100分-8位LED控制器(C/C++/Py/Java/Js/Go)

 

题目描述

有一个 8 位 LED 控制器，包含 8 个 LED 灯，编号 0~7，初始状态全灭，对应 8 位二进制为 00000000。
控制器可以接收以下三种指令：

  • Lx：点亮操作，x 表示 LED 编号 0~7，执行后将第 x 个 LED 置为 1
  • Dx：熄灭操作，x 表示 LED 编号 0~7，执行后将第 x 个 LED 置为 0
  • Tx：切换操作，x 表示 LED 编号 0~7，执行后将第 x 个 LED 状态取反，0 变 1，1 变 0

现在给定一组指令字符串，按顺序解析并执行所有指令，返回最终 8 位二进制对应的整数值。

例如：L0L1L2T1
表示先点亮 LED0、LED1、LED2，再切换 LED1，最终二进制为 00000101，对应整数 5。

输入描述

给定一组指令字符串，长度满足 0 <= 长度 <= 1000。

输出描述

返回最终 8 位二进制对应的整数值。

用例1

输入

L0L1L2T1

输出

5

题解

思路：模拟

  1. 创建一个长度为 8 的状态数组，初始全为 0。
  2. 按顺序每两个字符解析一条指令：
    • Lx：将第 x 位设为 1
    • Dx：将第 x 位设为 0
    • Tx：将第 x 位取反
  3. 全部执行完后，将最终 8 位二进制转换为十进制整数返回。

 

C++

#include<iostream>

#include<vector>

#include<string>

#include <utility>

#include <sstream>

#include<algorithm>

#include<cmath>

#include<map>

using namespace std;

 

int controlInstruct(string& instruct) {

    vector<int> bits(8, 0);

    for (int i = 0; i < instruct.size(); i+=2) {

        char c = instruct[i];

        int pos = instruct[i+1] - '0';

        // 点亮变为1

        if (c == 'L') {

            bits[pos] = 1;

        // 熄灭   

        } else if (c =='D') {

            bits[pos] = 0;

        // 切换   

        } else if (c == 'T'){

            bits[pos] ^= 1;

        }

    }

    int res = 0;

    for (int i = 7; i >= 0; i--) {

        res = res * 2 + bits[i];

    }

    return res;

}

 

int main() {

    string input;

    getline(cin, input);

    int res = controlInstruct(input);

    cout << res;

    return 0;

}

def controlInstruct(instruct):

    bits = [0] * 8

 

    i = 0

    while i < len(instruct):

        c = instruct[i]

        pos = int(instruct[i + 1])

 

        # 点亮变为1

        if c == 'L':

            bits[pos] = 1

        # 熄灭

        elif c == 'D':

            bits[pos] = 0

        # 切换

        elif c == 'T':

            bits[pos] ^= 1

 

        i += 2

 

    res = 0

    for i in range(7, -1, -1):

        res = res * 2 + bits[i]
